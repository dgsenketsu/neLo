Obiective

Aceasta aplicatie web este destinata complexelor hoteliere care doresc sa ofere utilizatorilor o platforma usor de folosit, unde pot afla diverse aspecte legate de hotel, precum facilitati si preturi si pot sa isi rezerva fara batai de cap vacanta visurilor lor.

Functionalitate

Prin intermediul acestei platforme clientul poate consulta real-time gradul de ocupare a hotelului pe o perioada de 3 luni, poate lua usor legatura cu personalul hotelului, poate sa vizualizeze o galerie foto cu imagini despre hotel, facilitati, preturi etc. Pe langa asta, clientul poate sa opteze sa-si rezerve o camera de hotel, procesul fiind mult simplificat si facut cu gandul la client si la timpul acestuia. In functie de numarul de zile dorite si de camera aleasa, clientului ii este prezentat toate optiunile disponibile in luna dorita.

Descrierea codului

config.php

Fisier de configurare, contine datele necesare conectarii la baza de date. A se modifica cu datele hostului.

Live acomodation module

check.php - Script PHP ce contine parte din modulul de actualizare in timp real al stadiului de ocupare al hotelului

webiste.js - Partea frontend al acestui script, utilizeaza ajax pentru a comunica cu serverul.

Exemplu Request Ajax

var request = $.ajax({ type: 'POST', url: 'check.php', dataType: 'text', });

send_form.php

Script PHP ce este folosit atunci cand clientul doreste sa ia legatura cu hotelul, preia datele clientului si le insereaza in baza de date

promo.js - Script auxiliar frontend ce contine warning destinate clientului

Reservation Module

reservation.php - Pagina principala ce incorporeaza scripturile JS pentru partea interactiva a procesului de rezervare

checkreservation.php - Preia preferintele clientului legate de data rezervarii si returneaza rezultate filtrare

submit_reservation.php - Inregistrarea rezervarii in baza de date

reserve.js - Partea frontend a modului, se ocupa cu actualizarii paginii in timp real pe masura ce clientul face rezervarea

Administration Module

login.php - Pagina de login

verify.php - Script PHP ce confirma identitatea utilizatorului care doreste sa se logheze.

panel.php+panel.js - Panoul de administrate al aplicatiei

panel-api.php - Script PHP ce ofera functionalitate panoului
